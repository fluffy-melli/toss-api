
> [!WARNING]
> 이 프로젝트는 **Toss Payments 비공식 라이브러리** 입니다.

Toss Payments API의 Go 클라이언트 라이브러리입니다. 일반결제, 자동결제, 거래내역 조회, 정산내역 조회 등 주요 기능을 제공합니다.

```sh
go get -u github.com/fluffy-melli/toss-api
```
